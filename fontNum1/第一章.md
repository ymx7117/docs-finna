# 第一章 产品概览
## 1.平台简介与定位
**Finna 大模型应用服务平台**是一个面向企业级客户打造的 Model-as-a-Service 平台，提供**统一的大模型能力接入、低门槛的智能应用开发工具、灵活的插件生态**，帮助企业快速构建多样化的 AI 应用，实现业务智能化升级。  
平台面向 **AI 开发者、业务专家、系统集成商** 等多类用户，支持 SaaS 与私有化部署，覆盖从模型调用、知识增强、流程编排，到多模态内容生成的全流程能力，适用于金融、政务、能源、制造等多个行业的智能化场景落地。
## 2.核心能力架构：1 + 4 模块
Finna 平台基于“1 个统一平台 + 4 大核心能力模块”构建：
- **统一平台**  
作为一站式平台，Finna 提供统一入口、用户空间、权限控制、资源管理与部署模式选择，支持企业级多项目协同与合规要求。  
- **四大能力模块**  
  -  模型广场：
聚合主流运营商通用、代码、多模态、语音等多类型模型，便于用户快速选择和切换适配模型。
  -  数据工坊：
构建企业级知识库，配合 RAG 检索增强能力提升问答质量，支持文档导入与分段处理。
  -  工具百宝箱：
  提供插件系统，支持连接第三方系统（如搜索、图谱、数据库、API 服务）扩展模型功能。
  -  计算平台：
支持模型训练、调参、推理优化等任务，统一管理算力资源及资源的可视化监控。
Finna 适用于多种业务场景，如智能客服、业务助手、投顾分析、内容生成、流程自动化等。
## 3.技术规格与部署方式
| 分类 | 能力说明 |
|:---:|---------|
| 访问方式 | 支持 Web 控制台与 API 双通道接入 |
| 部署模式 | 支持公有云 SaaS 快速部署，也支持本地私有化（支持 Docker   Compose 启动） |
| 安全能力 | 多租户数据隔离、精细化权限控制、审计日志、安全加密传输 |
| 系统能力 | 高可用架构，支持弹性伸缩与并发高访问，保障业务连续性 |
